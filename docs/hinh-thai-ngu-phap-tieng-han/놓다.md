# 놓다
{: .no_toc }

## Phạm trù

Động từ

## Cấu tạo

Kết hợp với '-어(아,여)' hoặc '-(이)라' dùng làm trợ động từ.

## Ý nghĩa

Diễn tả sự sự bảo lưu sự việc hoặc duy trì trạng thái.
